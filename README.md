# 🌌 SkillKode - The Nexus Patch (v3.0.0)

Welcome to **SkillKode**, a high-performance, cyberpunk-infused gamification platform built for developers who live on the edge of the digital frontier.

---

## 🛰️ Core Features

### 🌳 Neural Skill Tree
Progress beyond simple levels. Earn **SP (Skill Points)** to activate neural nodes that permanently augment your profile:
- **The Architect**: 25% XP multiplier.
- **Merchant Lord**: 50% Credit (CR) boost.
- **Precision Strike**: Critical hit logic for Raid Bosses.
- **Dark Web Access**: Unlocks the hidden Terminal marketplace.

### ⚔️ Nexus Arena (1v1 Code Duels)
Test your knowledge in real-time programming battles.
- Fight against AI opponents like **Dr0id** and **CyberShark**.
- Predict code outputs for **JavaScript**, **Python**, and **CSS**.
- Receive **Theoretical Analysis** after every round to master the "Why" behind the "What."

### 👹 Community Raid Boss
Join the global effort to take down **THE VOID OVERLORD**. 
- A shared health bar on your dashboard.
- Every strike from every user contributes to the objective.
- Earn **Micro-Credits** and **XP Buffers** with every hit.

### 🌡️ Dynamic Atmosphere
The UI breathes with your real-world clock. The theme colors shift automatically between:
- **Horizon** (Morning)
- **Solar** (Noon)
- **Sunset** (Evening)
- **Midnight** (Night)

---

## 💻 Technical Stack
- **Framework**: React 18 / Vite
- **Styling**: Tailwind CSS (Custom Glassmorphism + CRT Shaders)
- **State**: React Context API (Custom User Logic)
- **Animation**: Framer Motion (Spring Physics + 3D Gallery)
- **Audio**: Custom Synth SFX Engine

---

## ⌨️ Advanced Operations (Easter Eggs)

### 🧛 The Quantum Console
Press **`Alt + T`** to open the Command Layer.
- `scan`: Locate local sector vulnerabilities.
- `breach [host]`: Initiate a real-time brute force simulation.
- `market`: Access the secret Dark Web shop (Requires node activation).
- `system`: View hardware diagnostics.

## 🚀 Getting Started

1. **Install Dependencies**: `npm install`
2. **Launch Nexus**: `npm run dev`
3. **Login**: Enjoy the high-fidelity video background entrance.
